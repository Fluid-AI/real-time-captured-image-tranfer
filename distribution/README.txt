1. copy this whole folder
2. To change the port go to application.yaml and change the port to your desired port.
3. In linux or windows 'cd' to this directory only otherwise image folder will be created anywhere.
4. to run the jar simply put the following command :
	java -jar real-time-captured-image-transfer.jar &> log.txt &
5. the abpve command will run the jar in background.
